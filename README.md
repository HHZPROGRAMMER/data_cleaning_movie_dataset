# 🎬 Movies Data Analytics Project

Ushbu loyihada filmlar ma'lumotlar to'plami (Movies Dataset) Pandas va Matplotlib yordamida tahlil qilindi, tozalandi va yakuniy dashboard Power BI dasturida yaratildi.

## 📁 Loyiha Strukturasi
* `data/` - Asl va tozalangan CSV ma'lumotlar
* `notebooks/` - Ma'lumotlarni tozalash va EDA jarayoni (Jupyter Notebook)
* `images/` - Matplotlib vizualizatsiyalari
* `dashboard/` - Power BI fayli va skrinshoti
